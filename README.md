# Chatbot_DUK_Admission_Assistant
A Hybrid RAG Chatbot for Digital University Kerala (DUK) Admission 2025. Built with Llama 3.2 3B, LangChain, and Streamlit. Utilizes sentence-transformers/all-MiniLM-L6-v2 for efficient semantic retrieval, paired with a deterministic guardrail layer to ensure zero hallucinations on critical admission rules.
